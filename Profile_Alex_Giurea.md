Projects (website format)

Project 1
DrawGen
Professional tournament draws and court scheduling for tennis organizers

Short description
DrawGen is a Next.js web app that lets club directors, tennis coaches, and event managers create, schedule, edit, and export tournament draws in seconds, without spreadsheets or expensive software.

Target users

Club directors

Tennis coaches

Event managers

Tournament organizers

Key features

Guided wizard: tournament setup, entrant entry, seeding, and review in a few steps

Multiple formats: knockout (8, 16, 32, 64), round robin (4 to 12 players), qualifying, and consolation draws

Singles and doubles: full support for both formats

Scheduling studio: round by round date, time, court, and interval controls with live preview

Editable output: swap players, update winners and scores, adjust schedule labels directly on the draw

Cortex table view: courts as columns schedule layout with date filtering

Export and print: PDF, PNG, JPEG, and print ready outputs

Dashboard workspace: saved draws, folders, and resume from last route

AI assistant: context aware guidance for scheduling and draw planning

Tech stack

Framework: Next.js 16 (App Router), React 19, TypeScript

Styling: Tailwind CSS

Bracket rendering: Bracketry

Export: html to image plus jsPDF

AI: OpenAI API (optional, for assistant)

Repository
https://github.com/AlexGiurea/drawgen
