# Cinematic Portfolio — Alex Giurea

A high-fidelity, interactive personal portfolio website built with React, Tailwind CSS, and GSAP. Designed as a digital instrument with cinematic transitions, immersive micro-interactions, and a premium "Midnight Luxe" dark editorial aesthetic.

## 🚀 Features

- **Immersive GSAP Animations:** Purposeful scroll-triggered animations and seamless page transitions.
- **Magnetic Micro-Interactions:** Custom bouncy, magnetic button and link interactions for a premium feel.
- **Responsive "Midnight Luxe" Design:** A dark mode exclusive design system using Obsidian, Champagne, Ivory, and Slate, ensuring perfect display across all devices.
- **Interactive UI Components:** Includes a 3D image gallery, typewriter telemetry effects, and interactive scheduling components.
- **Optimized Performance:** Built on top of Vite and React 19 for blazing-fast load times.

## 🛠️ Tech Stack

- **Framework:** React 19
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (v3.4.17)
- **Animation:** GSAP 3 (with ScrollTrigger & ScrollToPlugin)
- **Icons:** Lucide React

## 🚀 Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AlexGiurea/cinematic-portfolio.git
   cd cinematic-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🎨 Design Philosophy

This project aims to eradicate "generic AI patterns" and build a site that feels like a functional artifact. Every scroll and animation is weighted and intentional.
